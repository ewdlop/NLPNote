## Organization Member Permissions

### Base Permission Levels
- **None**: No access by default
- **Read**: Can view and clone repositories
- **Write**: Can push to repositories
- **Admin**: Full access to all repositories

### Repository-Specific Roles
- **Read**: View and clone
- **Triage**: Manage issues and PRs (no code access)
- **Write**: Push to repository
- **Maintain**: Manage repository settings
- **Admin**: Full repository control
